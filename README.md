# 📊 Sorting Visualizer

This project is a web-based Sorting Visualizer developed using HTML, CSS, and JavaScript. It helps users understand how different sorting algorithms work through interactive animations and visual representation.

The application visually demonstrates the step-by-step process of sorting data, making it easier to grasp algorithm behavior and performance.

---

## 🚀 Features

- 📊 Visual representation of array elements using bars  
- 🎥 Real-time animation of sorting algorithms  
- 🔄 Multiple sorting algorithms implemented:
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - (Optional) Merge Sort / Quick Sort  
- ⚡ Adjustable speed of visualization  
- 🔢 Generate new random arrays  
- 📱 Fully responsive design  
- 🎨 Clean and interactive UI  

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling and animations  
- **JavaScript (ES6)** – Sorting logic and DOM manipulation

## ⚙️ How It Works

- Random bars are generated to represent array elements.
- Each bar height corresponds to a numerical value.
- When a sorting algorithm is selected:
  - The algorithm runs step-by-step.
  - Comparisons and swaps are visualized through animations.
- Users can observe how the array gets sorted in real-time.
